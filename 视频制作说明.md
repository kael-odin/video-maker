# 视频制作说明

## 🎬 如何生成视频

### 方法1: 使用渲染脚本（推荐）

```powershell
# 进入项目目录
cd e:/video/video-podcast-maker

# 渲染视频（选择一个质量级别）
.\render.ps1 preview   # 快速预览（1-2分钟）
.\render.ps1 standard # 标准质量（5-10分钟）
.\render.ps1 high     # 高质量（15-30分钟）
```

### 方法2: 手动渲染

```powershell
# 渲染4K横屏视频
npx remotion render src/remotion/index.ts MyVideo out/how-to-learn-programming-4k.mp4 `
  --public-dir projects/how-to-learn-programming/ `
  --codec h264 --quality 100 --crf 18
```

### 方法3: 使用Remotion Studio预览

```powershell
npx remotion studio src/remotion/index.ts --public-dir projects/how-to-learn-programming/
```

## 📊 渲染时间估算

- Preview: 1-2分钟
- Standard: 5-10分钟
- High: 15-30分钟

## 🎨 自定义样式

在Studio右侧面板调整：
- 颜色（主色调、背景色、文字颜色）
- 字体大小
- 动画效果

## 📁 输出文件

```
out/
├── how-to-learn-programming-4k.mp4      # 4K横屏视频
├── how-to-learn-programming-vertical.mp4 # 竖屏视频
└── thumbnail-*.png                      # 缩略图
```
